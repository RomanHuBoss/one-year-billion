Workflow

Крупные вехи запуска

Все шаги идут строго сверху вниз. Заблокированные gate нельзя обойти через страницу.

Fail-closed

Что мешает запуску

Причины берутся из backend runtime/preflight. `blocked` — штатное безопасное состояние.

Операционный центр

Следующее действие

Команды запускаются только через backend allowlist, без произвольного shell-доступа браузера.

Symbols

Символы

Единый источник статуса — backend `status_effective`. Таблица и карточка используют одни данные.

Traceability

Детали символа

Причины, trace_id и разрешенные safe-actions.

Выберите символ слева.

Safe actions

Безопасные действия оператора

Разрешены только действия, которые снижают риск или не увеличивают exposure. Запуск идет через backend /api/actions, с OPERATOR_API_KEY, reason и idempotency key.

Hard gates

Инварианты, которые нельзя обойти

Они enforced backend/DB/execution-контуром, а не внешним видом страницы.
